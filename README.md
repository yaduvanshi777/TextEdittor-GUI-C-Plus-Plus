## How to Run on Ubuntu/MacOS
### installing dependency 
```
sudo apt search libwxgtk3.0-gtk3-dev
```
### compiling :
```
g++ *.cpp `wx-config --cxxflags --libs`
```

### running 
```
./a.out
```
